Implementing a Queue with a Linked List

A queue is most naturally implemented with a singly-linked list that maintains pointers to both the head (front) and the tail (back). Enqueue appends to the tail; dequeue removes from the head. Both are O(1).

Node and Class Structure

LinkedListQueue outline
public class LinkedListQueue<T> {
    private class Node {
        T data;
        Node next;
        Node(T d) { data = d; }
    }

    private Node head;   // front of queue
    private Node tail;   // back of queue
    private int length;

    public LinkedListQueue() {
        head = tail = null;
        length = 0;
    }
}

Enqueue Operation

Create a new node. If the queue is empty, both head and tail point to it. Otherwise, link the current tail to the new node and advance tail.

enqueue()
public void enqueue(T item) {
    Node newNode = new Node(item);
    if (isEmpty()) {
        head = tail = newNode;
    } else {
        tail.next = newNode;  // link old tail to new node
        tail = newNode;       // advance tail
    }
    length++;
}

Dequeue Operation

Save the head's data, advance head to the next node, and return the saved data. If the queue becomes empty, reset tail to null as well.

dequeue()
public T dequeue() {
    if (isEmpty()) throw new RuntimeException("Queue is empty.");
    T val = head.data;
    head  = head.next;
    if (head == null) tail = null;  // queue is now empty
    length--;
    return val;
}

Full Implementation and Demo

LinkedListQueue.java
public class LinkedListQueue<T> {
    private class Node { T data; Node next; Node(T d){data=d;} }

    private Node head, tail;
    private int length;

    public boolean isEmpty()   { return length == 0; }
    public int     getLength() { return length; }
    public T       peek()      {
        if (isEmpty()) throw new RuntimeException("Queue is empty.");
        return head.data;
    }

    public void enqueue(T item) {
        Node n = new Node(item);
        if (isEmpty()) { head = tail = n; }
        else { tail.next = n; tail = n; }
        length++;
    }

    public T dequeue() {
        if (isEmpty()) throw new RuntimeException("Queue is empty.");
        T val = head.data;
        head  = head.next;
        if (head == null) tail = null;
        length--;
        return val;
    }

    public static void main(String[] args) {
        LinkedListQueue<String> q = new LinkedListQueue<>();
        q.enqueue("Alice");
        q.enqueue("Bob");
        q.enqueue("Carol");

        System.out.println("Peek: "   + q.peek());      // Alice
        System.out.println("Remove: " + q.dequeue());   // Alice
        System.out.println("Remove: " + q.dequeue());   // Bob
        System.out.println("Size: "   + q.getLength()); // 1
    }
}
