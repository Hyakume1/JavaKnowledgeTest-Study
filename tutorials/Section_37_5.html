Building a Queue With a Linked List

A queue works naturally with a linked list because we need fast access to both ends — the front (for removing) and the back (for adding). So we keep two pointers: head points to the front of the line, and tail points to the back.

  • Enqueue — add a new node at the tail end (back of the line)
  • Dequeue — remove the node at the head (front of the line)

Node and Class Structure

LinkedListQueue outline
public class LinkedListQueue<T> {
    private class Node {
        T data;
        Node next;
        Node(T d) { data = d; }
    }

    private Node head;   // front of queue (dequeue from here)
    private Node tail;   // back of queue (enqueue here)
    private int length;

    public LinkedListQueue() {
        head = tail = null;
        length = 0;
    }
}

Enqueue: Adding to the Back

Create a new node. If the queue is empty, both head and tail point to it. Otherwise, link the current tail to the new node and move the tail pointer forward.

enqueue()
public void enqueue(T item) {
    Node newNode = new Node(item);
    if (isEmpty()) {
        head = tail = newNode;       // first item — head and tail both point to it
    } else {
        tail.next = newNode;         // link old tail to new node
        tail = newNode;              // new node is now the tail
    }
    length++;
}

Dequeue: Removing From the Front

Save the head node's value, move the head pointer to the next node, and return the saved value. If the queue becomes empty (head becomes null), reset tail to null too.

dequeue()
public T dequeue() {
    if (isEmpty()) throw new RuntimeException("Queue is empty.");
    T val = head.data;
    head  = head.next;
    if (head == null) tail = null;  // queue is now empty — clean up tail
    length--;
    return val;
}

Full Implementation and Demo

LinkedListQueue.java
public class LinkedListQueue<T> {
    private class Node { T data; Node next; Node(T d){data=d;} }

    private Node head, tail;
    private int length;

    public boolean isEmpty()   { return length == 0; }
    public int     getLength() { return length; }
    public T       peek()      {
        if (isEmpty()) throw new RuntimeException("Queue is empty.");
        return head.data;
    }

    public void enqueue(T item) {
        Node n = new Node(item);
        if (isEmpty()) { head = tail = n; }
        else { tail.next = n; tail = n; }
        length++;
    }

    public T dequeue() {
        if (isEmpty()) throw new RuntimeException("Queue is empty.");
        T val = head.data;
        head  = head.next;
        if (head == null) tail = null;
        length--;
        return val;
    }

    public static void main(String[] args) {
        LinkedListQueue<String> q = new LinkedListQueue<>();
        q.enqueue("Alice");
        q.enqueue("Bob");
        q.enqueue("Carol");

        System.out.println("Peek: "   + q.peek());      // Alice
        System.out.println("Remove: " + q.dequeue());   // Alice
        System.out.println("Remove: " + q.dequeue());   // Bob
        System.out.println("Size: "   + q.getLength()); // 1
    }
}
