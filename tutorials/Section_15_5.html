Writing to a File

To write to a file, create a FileOutputStream pointing to the output file, then wrap it with a PrintWriter to get familiar methods like print(), println(), and printf().

If the file does not yet exist it is created. If it already exists, it is overwritten. The constructor throws an exception if the file cannot be created (e.g., permission denied).

Basic write pattern
import java.io.FileOutputStream;
import java.io.PrintWriter;
import java.io.IOException;

FileOutputStream fileStream = new FileOutputStream("output.txt");
PrintWriter outFS = new PrintWriter(fileStream);

outFS.println("Hello, file!");
outFS.printf("Value: %d%n", 42);

outFS.close();   // IMPORTANT: flushes the buffer and closes the file

Always call close(). If you forget, the buffer may not be flushed and the file may be incomplete or empty.

Example: Writing a Grade Report

GradeReport.java
import java.util.Scanner;
import java.io.FileOutputStream;
import java.io.PrintWriter;
import java.io.IOException;

public class GradeReport {
    public static void main(String[] args) throws IOException {
        Scanner scnr = new Scanner(System.in);

        System.out.print("Output file name: ");
        String fileName = scnr.next();

        FileOutputStream fileStream = new FileOutputStream(fileName);
        PrintWriter outFS = new PrintWriter(fileStream);

        outFS.println("Student Grade Report");
        outFS.println("====================");

        double total = 0;
        int count = 0;

        System.out.println("Enter grades (-1 to stop):");
        double grade = scnr.nextDouble();

        while (grade != -1) {
            outFS.printf("Grade %d: %.1f%n", count + 1, grade);
            total += grade;
            count++;
            grade = scnr.nextDouble();
        }

        double avg = (count > 0) ? total / count : 0;
        outFS.printf("Average: %.2f%n", avg);

        outFS.close();
        System.out.println("Report written to " + fileName);
    }
}

Appending to a File

By default, FileOutputStream overwrites the file. To append instead, pass true as the second argument:

Append mode
FileOutputStream fileStream = new FileOutputStream("log.txt", true);
PrintWriter outFS = new PrintWriter(fileStream);
outFS.println("New log entry");
outFS.close();

Writing an HTML File

Since HTML is plain text, you write it just like any other text file.

Writing a simple HTML page
FileOutputStream fileStream = new FileOutputStream("page.html");
PrintWriter outFS = new PrintWriter(fileStream);

outFS.println("<html><body>");
outFS.println("<h1>Hello from Java!</h1>");
outFS.println("</body></html>");

outFS.close();
