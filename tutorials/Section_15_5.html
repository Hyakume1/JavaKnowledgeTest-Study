Writing to a File

Just like you can read data from a file, you can also save your program's output to a file. Instead of printing to the screen, you print to a file. This is great for creating reports or saving results for later.

To write to a file, you need two things working together:

  1. A FileOutputStream — points to the file you want to write to.
  2. A PrintWriter wrapped around it — gives you familiar methods like println() and printf() but they write to the file instead of the screen.

Watch out: If the file already exists, it gets completely overwritten (erased and replaced). If it doesn't exist, Java creates it for you.

Basic write pattern
import java.io.FileOutputStream;
import java.io.PrintWriter;
import java.io.IOException;

FileOutputStream fileStream = new FileOutputStream("output.txt");
PrintWriter outFS = new PrintWriter(fileStream);

outFS.println("Hello, file!");
outFS.printf("Value: %d%n", 42);

outFS.close();   // VERY IMPORTANT: always close the file when done!

Always call close() when you are done writing! If you skip this step, Java might not finish saving everything to the file, and you could end up with a missing or incomplete file.

Example: Writing a Grade Report

GradeReport.java
import java.util.Scanner;
import java.io.FileOutputStream;
import java.io.PrintWriter;
import java.io.IOException;

public class GradeReport {
    public static void main(String[] args) throws IOException {
        Scanner scnr = new Scanner(System.in);

        System.out.print("Output file name: ");
        String fileName = scnr.next();

        FileOutputStream fileStream = new FileOutputStream(fileName);
        PrintWriter outFS = new PrintWriter(fileStream);

        outFS.println("Student Grade Report");
        outFS.println("====================");

        double total = 0;
        int count = 0;

        System.out.println("Enter grades (-1 to stop):");
        double grade = scnr.nextDouble();

        while (grade != -1) {
            outFS.printf("Grade %d: %.1f%n", count + 1, grade);
            total += grade;
            count++;
            grade = scnr.nextDouble();
        }

        double avg = (count > 0) ? total / count : 0;
        outFS.printf("Average: %.2f%n", avg);

        outFS.close();
        System.out.println("Report written to " + fileName);
    }
}

Appending Instead of Overwriting

By default, opening a file for writing erases everything already in it. If you want to add to the end of an existing file instead (like writing a new entry in a log), pass true as the second argument to FileOutputStream:

Append mode — adds to the file instead of erasing it
FileOutputStream fileStream = new FileOutputStream("log.txt", true);
PrintWriter outFS = new PrintWriter(fileStream);
outFS.println("New log entry");
outFS.close();

Writing an HTML File

Since an HTML file is just plain text, you can write one the exact same way:

Writing a simple HTML page
FileOutputStream fileStream = new FileOutputStream("page.html");
PrintWriter outFS = new PrintWriter(fileStream);

outFS.println("<html><body>");
outFS.println("<h1>Hello from Java!</h1>");
outFS.println("</body></html>");

outFS.close();
