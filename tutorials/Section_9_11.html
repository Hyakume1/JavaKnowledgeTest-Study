ArrayList Overview

An ArrayList is a resizable array from the Java Collections Framework. Unlike a regular array, an ArrayList automatically grows and shrinks as elements are added or removed. Import java.util.ArrayList to use it.

Declaration and creation
ArrayList<Integer> nums  = new ArrayList<Integer>();
ArrayList<String>  names = new ArrayList<String>();

Core Methods

Assume teamNums is an ArrayList<Integer> currently holding: 5, 9, 23, 11, 14.

Common operations
teamNums.size()         // → 5  (number of elements)
teamNums.isEmpty()      // → false
teamNums.get(2)         // → 23 (zero-based index)
teamNums.set(0, 99)     // replaces index 0 with 99, returns old value 5
teamNums.add(77)        // appends 77 at the end
teamNums.add(1, 88)     // inserts 88 at index 1, shifts rest right
teamNums.remove(3)      // removes element at index 3, shifts rest left
teamNums.clear()        // removes all elements; size → 0

remove(int) vs remove(Object)

remove(int index) deletes by position. To delete by value from an ArrayList<Integer>, wrap the value: list.remove(Integer.valueOf(5)). Calling list.remove(5) removes the element at index 5, not the element whose value is 5.

Iterating an ArrayList

Two loop patterns
// Index-based — use when you need the index
for (int i = 0; i < teamNums.size(); i++) {
    System.out.println(teamNums.get(i));
}

// Enhanced for — cleaner when you only need the value
for (int val : teamNums) {
    System.out.println(val);
}

Example: Player Roster Manager

PlayerManager.java
import java.util.ArrayList;
import java.util.Scanner;

public class PlayerManager {
    private ArrayList<Integer> players = new ArrayList<Integer>();

    public void addPlayer(int num) {
        players.add(num);
    }

    public void deletePlayer(int num) {
        for (int i = 0; i < players.size(); i++) {
            if (players.get(i).equals(num)) {
                players.remove(i);
                return;
            }
        }
    }

    public void printPlayers() {
        for (int n : players) {
            System.out.println(" " + n);
        }
    }

    public static void main(String[] args) {
        Scanner scnr = new Scanner(System.in);
        PlayerManager roster = new PlayerManager();
        String cmd = "-";
        System.out.println("Commands: a=add  d=delete  p=print  q=quit");
        while (!cmd.equals("q")) {
            System.out.print("Command: ");
            cmd = scnr.next();
            if (cmd.equals("a")) {
                System.out.print("Player number: ");
                roster.addPlayer(scnr.nextInt());
            } else if (cmd.equals("d")) {
                System.out.print("Player number: ");
                roster.deletePlayer(scnr.nextInt());
            } else if (cmd.equals("p")) {
                roster.printPlayers();
            }
        }
    }
}

Key Points

  • ArrayList indices start at 0, just like arrays.
  • Accessing an out-of-bounds index throws IndexOutOfBoundsException.
  • ArrayList only stores objects — use wrapper types (Integer, Double, String), not primitives.
  • When deleting elements inside a forward loop, iterate backwards or use an iterator to avoid skipping elements.
