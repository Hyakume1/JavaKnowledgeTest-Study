Reading Words with Scanner

The next() method of Scanner reads the next whitespace-delimited token. Whitespace — spaces, tabs, and newlines — is consumed but not included in the returned string. Everything else is part of the word, including punctuation.

next() includes punctuation
Scanner scnr = new Scanner(System.in);
// Input: "Hello, world."
String w1 = scnr.next();  // "Hello,"  ← comma is included
String w2 = scnr.next();  // "world."  ← period is included

To read only letters (stripping punctuation and numbers), call useDelimiter() with a pattern that treats non-letters as separators:

useDelimiter to skip non-letters
Scanner scnr = new Scanner(new FileInputStream("story.txt"));
scnr.useDelimiter("[^a-zA-Z]+");  // any non-letter sequence is a delimiter
while (scnr.hasNext()) {
    System.out.println(scnr.next()); // only letters, no punctuation
}

Reading One Character at a Time

Pass an empty string to useDelimiter("") so that every call to next() returns a single character as a one-character string.

Character-by-character reading
Scanner scnr = new Scanner(new FileInputStream("input.txt"));
scnr.useDelimiter("");  // each next() returns one character

while (scnr.hasNext()) {
    char ch = scnr.next().charAt(0);
    System.out.print(ch);
}

The Character Class — Classifying Characters

The Character class provides static methods to test what kind of character you have:

  • Character.isDigit(ch) — true if '0'–'9'
  • Character.isLetter(ch) — true if alphabetic
  • Character.isUpperCase(ch) — true if uppercase letter
  • Character.isLowerCase(ch) — true if lowercase letter
  • Character.isWhitespace(ch) — true if space, tab, or newline
Classifying characters
char ch = 'A';
System.out.println(Character.isLetter(ch));    // true
System.out.println(Character.isDigit(ch));     // false
System.out.println(Character.isUpperCase(ch)); // true

Reading Lines with nextLine()

When each line of a file is a complete record, use nextLine() to read the entire line at once, then parse it further.

Reading a country/population file line by line
// File format: "Brazil 214300000"
Scanner inFS = new Scanner(new FileInputStream("countries.txt"));
while (inFS.hasNextLine()) {
    String line = inFS.nextLine();
    // Find where the number starts
    int i = 0;
    while (i < line.length() && !Character.isDigit(line.charAt(i))) {
        i++;
    }
    String country = line.substring(0, i).trim();
    long   pop     = Long.parseLong(line.substring(i).trim());
    System.out.printf("%-20s %,d%n", country, pop);
}
Output 
Brazil               214,300,000

Converting Strings to Numbers

Use the static parse methods to convert a numeric string to a primitive value:

Parsing strings
String s1 = "42";
String s2 = "3.14";

int    n = Integer.parseInt(s1);      // 42
double d = Double.parseDouble(s2);    // 3.14

// Throws NumberFormatException if the string is not a valid number:
int bad = Integer.parseInt("12abc"); // EXCEPTION

Formatted Output with printf()

When printing tables, use %-Ns (left-align string in N characters) and %N.2f (right-align float with 2 decimal places):

Aligned item-price table
String[] items  = {"Coffee", "Bagel",  "Juice"};
double[] prices = {2.50,     1.75,     3.00};
double total = 0;

for (int i = 0; i < items.length; i++) {
    System.out.printf("%-12s %6.2f%n", items[i] + ":", prices[i]);
    total += prices[i];
}
System.out.printf("%-12s %6.2f%n", "Total:", total);
Output 
Coffee:        2.50
Bagel:         1.75
Juice:         3.00
Total:         7.25
