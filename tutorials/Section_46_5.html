How Scanner Reads Words

When you call next() on a Scanner, it skips any spaces, tabs, or newlines, then reads everything up to the next space, tab, or newline. So if the input is "Hello, world.", you get two tokens: "Hello," and "world." — comma and period included! Punctuation is NOT automatically stripped.

next() includes punctuation
Scanner scnr = new Scanner(System.in);
// Input: "Hello, world."
String w1 = scnr.next();  // "Hello,"  ← comma is part of the token
String w2 = scnr.next();  // "world."  ← period is part of the token

Reading Only Letters (No Punctuation)

If you want to read only the letters and skip all punctuation and numbers, you can change what Scanner treats as a separator using useDelimiter(). The pattern [^a-zA-Z]+ means "any character that is NOT a letter counts as a separator."

useDelimiter to skip non-letter characters
Scanner scnr = new Scanner(new FileInputStream("story.txt"));
scnr.useDelimiter("[^a-zA-Z]+");  // non-letters become separators
while (scnr.hasNext()) {
    System.out.println(scnr.next()); // prints only pure words, no punctuation
}

Reading One Character at a Time

Pass an empty string to useDelimiter("") and now every call to next() returns exactly one character. This is useful when you need to process text letter by letter.

Character-by-character reading
Scanner scnr = new Scanner(new FileInputStream("input.txt"));
scnr.useDelimiter("");  // each next() returns exactly one character

while (scnr.hasNext()) {
    char ch = scnr.next().charAt(0);
    System.out.print(ch);
}

Testing What Kind of Character You Have

Java's Character class has static helper methods to classify characters. You don't need to import anything — it's always available.

  • Character.isDigit(ch) — true if the character is '0' through '9'
  • Character.isLetter(ch) — true if the character is a letter (a-z or A-Z)
  • Character.isUpperCase(ch) — true if the character is an uppercase letter
  • Character.isLowerCase(ch) — true if the character is a lowercase letter
  • Character.isWhitespace(ch) — true if the character is a space, tab, or newline
Classifying characters
char ch = 'A';
System.out.println(Character.isLetter(ch));    // true
System.out.println(Character.isDigit(ch));     // false
System.out.println(Character.isUpperCase(ch)); // true

Reading an Entire Line at Once

When each line of a file is one complete record, use nextLine() to read the whole line, then break it apart yourself.

Reading a country/population file line by line
// File format: "Brazil 214300000"
Scanner inFS = new Scanner(new FileInputStream("countries.txt"));
while (inFS.hasNextLine()) {
    String line = inFS.nextLine();
    // Find where the number starts
    int i = 0;
    while (i < line.length() && !Character.isDigit(line.charAt(i))) {
        i++;
    }
    String country = line.substring(0, i).trim();
    long   pop     = Long.parseLong(line.substring(i).trim());
    System.out.printf("%-20s %,d%n", country, pop);
}
Output 
Brazil               214,300,000

Converting a String to a Number

When you read a number as a String (like "42"), you can't do math on it until you convert it. Use these static methods:

Parsing strings into numbers
String s1 = "42";
String s2 = "3.14";

int    n = Integer.parseInt(s1);      // 42
double d = Double.parseDouble(s2);    // 3.14

// If the string isn't a valid number, you get a NumberFormatException:
int bad = Integer.parseInt("12abc"); // CRASHES

Building a Formatted Table With printf()

Aligned item-price table
String[] items  = {"Coffee", "Bagel",  "Juice"};
double[] prices = {2.50,     1.75,     3.00};
double total = 0;

for (int i = 0; i < items.length; i++) {
    System.out.printf("%-12s %6.2f%n", items[i] + ":", prices[i]);
    total += prices[i];
}
System.out.printf("%-12s %6.2f%n", "Total:", total);
Output 
Coffee:        2.50
Bagel:         1.75
Juice:         3.00
Total:         7.25
