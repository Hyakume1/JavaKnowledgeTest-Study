Array-Based Queue

A queue can be implemented with an array. Two extra variables are required:

  • length — number of items currently in the queue
  • frontIndex — index of the front item

Items are not shifted when dequeuing. Instead, frontIndex advances forward (wrapping around when it reaches the end of the array). This creates a circular buffer.

Circular buffer concept
// Array size 6. After enqueue(A), enqueue(B), enqueue(C), dequeue():
// frontIndex = 1, length = 2
//  index: [ 0,  1,  2,  3,  4,  5 ]
//  data:  [ A,  B,  C,  -,  -,  - ]
//              ^front

// After enqueue(D), enqueue(E), enqueue(F), enqueue(G) (wraps around):
//  index: [ G,  B,  C,  D,  E,  F ]
//  frontIndex = 1, length = 6

Bounded vs Unbounded

  • A bounded queue uses a fixed maxLength. Enqueueing when full returns false.
  • An unbounded queue resizes (doubles the array) when full. After resizing, existing items are copied so that frontIndex resets to 0.

Enqueue and Dequeue Logic

Enqueue and dequeue using modular arithmetic
// Enqueue: compute the index for the new item
int newIndex = (frontIndex + length) % array.length;
array[newIndex] = item;
length++;

// Dequeue: return front item and advance frontIndex
T val = array[frontIndex];
frontIndex = (frontIndex + 1) % array.length;
length--;
return val;

Example: Bounded Queue with Capacity 4

Step-by-step trace
ArrayQueue q = new ArrayQueue(4);  // max 4 items
q.enqueue("A");   // [A, -, -, -]  front=0 length=1
q.enqueue("B");   // [A, B, -, -]  front=0 length=2
q.enqueue("C");   // [A, B, C, -]  front=0 length=3
q.dequeue();      // returns "A"   [-, B, C, -]  front=1 length=2
q.enqueue("D");   // [-, B, C, D]  front=1 length=3
q.enqueue("E");   // [E, B, C, D]  front=1 length=4  (E wraps to index 0)
q.enqueue("F");   // returns false — queue is full

Key Insight: Why Circular?

Without wrapping, dequeueing would leave empty slots at the front that can never be reused, wasting space. The modular arithmetic (frontIndex + length) % array.length lets the queue reuse those freed slots by wrapping around to the beginning of the array.
