Debugging Recursive Methods

Recursive methods can be difficult to debug because many calls are in progress simultaneously, each with its own local variables. A useful technique is to add print statements that show the current state at each call depth.

Indenting by Recursion Depth

Pass an extra indent string parameter through recursive calls. Each recursive call appends extra spaces to indent, so the output visually mirrors the call-stack depth.

NameFinder.java with debug output
import java.util.ArrayList;
import java.util.Scanner;

public class NameFinder {
    public static int findMatch(ArrayList<String> list, String target,
                                int low, int high, String indent) {
        int mid   = (low + high) / 2;
        int range = (high - low) + 1;

        System.out.println(indent + "Searching range [" + low + ", " + high + "]"
                           + ", mid = " + mid + " (" + list.get(mid) + ")");

        int result;
        if (target.equals(list.get(mid))) {
            System.out.println(indent + "→ Found!");
            result = mid;
        } else if (range == 1) {
            System.out.println(indent + "→ Not found.");
            result = -1;
        } else if (target.compareTo(list.get(mid)) < 0) {
            System.out.println(indent + "→ Go lower.");
            result = findMatch(list, target, low, mid, indent + "   ");
        } else {
            System.out.println(indent + "→ Go higher.");
            result = findMatch(list, target, mid + 1, high, indent + "   ");
        }

        System.out.println(indent + "Returning " + result);
        return result;
    }

    public static void main(String[] args) {
        ArrayList<String> attendees = new ArrayList<String>();
        attendees.add("Adams, Mary");
        attendees.add("Carver, Michael");
        attendees.add("Domer, Hugo");
        attendees.add("Fredericks, Carlos");
        attendees.add("Li, Jie");

        Scanner scnr = new Scanner(System.in);
        System.out.print("Enter name: ");
        String name = scnr.nextLine();

        int pos = findMatch(attendees, name, 0, attendees.size() - 1, "   ");
        System.out.println(pos >= 0 ? "Found at " + pos : "Not found.");
    }
}

Practical Debugging Tips

  • Print the parameters at the start of each call to confirm the problem is actually shrinking.
  • Print the return value before returning to trace values flowing back up the chain.
  • When done debugging, comment out the print statements — they also serve as documentation.
  • If recursion appears infinite, add a check: if the depth exceeds a reasonable limit, print a warning and return.
