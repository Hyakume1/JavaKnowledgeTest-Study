Array-Based Stack

A stack can be implemented with an array. The bottom of the stack lives at array[0] and the top at array[length - 1]. The variable length tracks how many items are currently in the stack.

Visual layout
// After push(5), push(10), push(15):
//  index:  0    1    2
//  array: [5,  10,  15,  ...]
//  length = 3        ^  top is here

Bounded vs Unbounded

  • A bounded stack has a maximum size equal to the array's initial length. Pushing onto a full stack is an error.
  • An unbounded stack doubles the array's size whenever it fills up (like ArrayList does internally).

ArrayStack Implementation

ArrayStack.java — supports both bounded and unbounded operation
import java.util.Arrays;

public class ArrayStack<T> {
    private Object[] array;
    private int length;
    private int maxLength;  // negative = unbounded

    // maxLength < 0 → unbounded; maxLength >= 0 → bounded
    public ArrayStack(int maxLength) {
        int initialCap = (maxLength < 0) ? 10 : maxLength;
        array     = new Object[initialCap];
        length    = 0;
        this.maxLength = maxLength;
    }

    public boolean isEmpty()   { return length == 0; }
    public int     getLength() { return length; }

    @SuppressWarnings("unchecked")
    public T peek() {
        if (isEmpty()) throw new RuntimeException("Stack underflow.");
        return (T) array[length - 1];
    }

    public boolean push(T item) {
        if (maxLength >= 0 && length == maxLength) {
            return false; // bounded stack is full
        }
        if (length == array.length) {
            // resize: double the capacity (unbounded only)
            array = Arrays.copyOf(array, array.length * 2);
        }
        array[length] = item;
        length++;
        return true;
    }

    @SuppressWarnings("unchecked")
    public T pop() {
        if (isEmpty()) throw new RuntimeException("Stack underflow.");
        T val = (T) array[length - 1];
        array[length - 1] = null; // help GC
        length--;
        return val;
    }

    public static void main(String[] args) {
        // Unbounded stack (pass negative max)
        ArrayStack<String> stack = new ArrayStack<>(-1);
        stack.push("alpha");
        stack.push("beta");
        stack.push("gamma");
        System.out.println(stack.peek());   // gamma
        System.out.println(stack.pop());    // gamma
        System.out.println(stack.pop());    // beta
        System.out.println("Size: " + stack.getLength()); // 1

        // Bounded stack with max 2
        ArrayStack<Integer> bounded = new ArrayStack<>(2);
        System.out.println(bounded.push(1)); // true
        System.out.println(bounded.push(2)); // true
        System.out.println(bounded.push(3)); // false (full)
    }
}
