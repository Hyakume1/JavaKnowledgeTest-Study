Reading Data From a File

So far you've read input that the user types. But what if all your data is already in a file? You can make your program read it automatically — no typing needed. This is super useful for large datasets.

To read from a file, you need two things working together:

  1. A FileInputStream — think of it as opening the file and pointing at it.
  2. A Scanner wrapped around it — this reads the contents the same way you read keyboard input.
File reading pattern
import java.util.Scanner;
import java.io.FileInputStream;
import java.io.IOException;

// main must declare: throws IOException
FileInputStream fileByteStream = new FileInputStream("data.txt");
Scanner inFS = new Scanner(fileByteStream);

int x = inFS.nextInt();       // read a whole number from the file
String s = inFS.next();       // read a word from the file
String line = inFS.nextLine();// read a full line from the file

fileByteStream.close();       // always close the file when you're done

Important: You must add throws IOException to your main() method when working with files. This is just Java's way of saying "I know the file might not exist — I acknowledge that risk." If the file isn't found, Java throws a FileNotFoundException.

Reading Until the End of the File

You usually don't know exactly how many items are in a file. Use hasNextInt() (or hasNext()) in a loop — it returns true as long as there's more data to read, and false when you've reached the end of the file.

FileReadVaryingAmount.java
import java.util.Scanner;
import java.io.FileInputStream;
import java.io.IOException;

public class FileReadVaryingAmount {
    public static void main(String[] args) throws IOException {
        FileInputStream fileByteStream = new FileInputStream("myfile.txt");
        Scanner inFS = new Scanner(fileByteStream);

        System.out.println("Reading numbers:");
        while (inFS.hasNextInt()) {
            int num = inFS.nextInt();
            System.out.println("num: " + num);
        }

        fileByteStream.close();
    }
}

Example: Counting Word Occurrences

CountingWords.java
import java.util.Scanner;
import java.io.FileInputStream;
import java.io.IOException;

public class CountingWords {
    public static void main(String[] args) throws IOException {
        Scanner scnr = new Scanner(System.in);
        FileInputStream fileByteStream = new FileInputStream("wordFile.txt");
        Scanner inFS = new Scanner(fileByteStream);

        System.out.print("Enter a word to count: ");
        String target = scnr.next();
        int count = 0;

        while (inFS.hasNext()) {
            if (inFS.next().equals(target)) {
                count++;
            }
        }

        System.out.println(target + " appears " + count + " time(s).");
        fileByteStream.close();
    }
}

Example: Restaurant Review Averages

ReviewSystem.java
import java.util.Scanner;
import java.io.FileInputStream;
import java.io.IOException;

public class ReviewSystem {
    public static void main(String[] args) throws IOException {
        FileInputStream fileByteStream = new FileInputStream("reviews.txt");
        Scanner inFS = new Scanner(fileByteStream);

        String restaurantName = inFS.nextLine();
        System.out.println(restaurantName);
        System.out.println("--------------------");

        int sum = 0, count = 0;
        while (inFS.hasNext()) {
            String user   = inFS.next();
            int    rating = inFS.nextInt();
            System.out.println(user + ": " + rating);
            sum += rating;
            count++;
        }

        System.out.println("--------------------");
        System.out.printf("Average rating: %.2f%n", (double) sum / count);
        fileByteStream.close();
    }
}
