Exploring All Possibilities With Recursion

Recursion is amazing at exploring every single possibility — every ordering of a list, every combination of items, every possible route. The pattern always looks the same:

  1. Make a choice (pick one item).
  2. Recurse with the remaining options.
  3. Undo the choice and try the next option.

All Permutations of a Word

A permutation is a different ordering of the same letters. "cat" has 6 permutations: cat, cta, act, atc, tca, tac. To generate them all recursively, at each step you pick one letter from the unused letters, add it to your built-up word, recurse, then put it back and try a different letter.

WordScrambler.java
import java.util.Scanner;

public class WordScrambler {
    // 'remaining' = letters not yet used, 'chosen' = letters placed so far
    public static void scramble(String remaining, String chosen) {
        if (remaining.length() == 0) {       // base case: all letters placed
            System.out.println(chosen);
        } else {                             // try placing each remaining letter
            for (int i = 0; i < remaining.length(); i++) {
                String letter = remaining.substring(i, i + 1);
                String rest   = remaining.substring(0, i) + remaining.substring(i + 1);
                scramble(rest, chosen + letter);
            }
        }
    }

    public static void main(String[] args) {
        Scanner scnr = new Scanner(System.in);
        System.out.print("Enter a word: ");
        String word = scnr.next();
        scramble(word, "");
    }
}

A word of N letters has N! permutations (3 letters = 6, 4 letters = 24, 8 letters = 40,320). Writing this with nested loops is basically impossible — recursion handles it elegantly.

All Combinations of K Items (Shopping Spree)

To find all 3-item combinations from a list: add one item to your bag, recurse with that item removed from the available pool, then remove it from the bag and try the next item.

Core pattern — pick 3 items from a list
public static void combinations(ArrayList<String> bag,
                                ArrayList<String> remaining,
                                int targetSize) {
    if (bag.size() == targetSize) {        // base case: bag is full — print it
        System.out.println(bag);
        return;
    }
    for (int i = 0; i < remaining.size(); i++) {
        String item = remaining.remove(i); // pick item
        bag.add(item);
        combinations(bag, remaining, targetSize);
        bag.remove(bag.size() - 1);        // un-pick item (backtrack)
        remaining.add(i, item);
    }
}

All Possible Routes (Traveling Salesman)

Find every possible route through a set of cities: visit one city, recurse on the unvisited cities, then "unvisit" and try the next one. The base case is when all cities have been visited.

TravelingSalesman.java (3 cities)
import java.util.ArrayList;

public class TravelingSalesman {
    static String[] cities = {"Boston", "Chicago", "Los Angeles"};
    static int[][] dist = {
        {0, 960, 2960},   // Boston to Chicago / LA
        {960, 0, 2011},   // Chicago to Boston / LA
        {2960, 2011, 0}   // LA to Boston / Chicago
    };

    public static void allPaths(ArrayList<Integer> path,
                                ArrayList<Integer> toVisit) {
        if (path.size() == cities.length) {    // base case: visited all cities
            int total = 0;
            for (int i = 1; i < path.size(); i++) {
                total += dist[path.get(i-1)][path.get(i)];
            }
            System.out.println(path.stream()
                .map(i -> cities[i]).toList() + " = " + total + " mi");
            return;
        }
        for (int i = 0; i < toVisit.size(); i++) {
            int city = toVisit.remove(i);
            path.add(city);
            allPaths(path, toVisit);
            path.remove(path.size() - 1);
            toVisit.add(i, city);
        }
    }

    public static void main(String[] args) {
        ArrayList<Integer> toVisit = new ArrayList<>();
        for (int i = 0; i < cities.length; i++) toVisit.add(i);
        allPaths(new ArrayList<>(), toVisit);
    }
}
