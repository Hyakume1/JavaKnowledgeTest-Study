printf() and Formatted Output

System.out.printf() (and the equivalent System.out.format()) lets you control exactly how values appear in output. The first argument is a format string containing plain text and format specifiers — placeholders that begin with %.

Basic syntax
System.out.printf("%-10s %5d %8.2f%n", "Item", 3, 19.99);
//                  ^       ^    ^  ^
//              left-pad  int  float  newline
Output 
Item            3    19.99

Common Format Specifiers

  • %d — integer (int, long)
  • %f — floating-point (double, float)
  • %s — String
  • %c — character
  • %b — boolean
  • %e — scientific notation (e.g. 1.23e+02)
  • %n — platform-appropriate newline (preferred over \n in printf)

Floating-Point Formatting

The sub-specifier .N controls decimal places. A width number before the decimal controls total field width.

Floating-point examples
double miles = 10.3;
double hoursFly   = miles / 500.0;
double hoursDrive = miles / 60.0;

System.out.printf("%.2f miles would take:%n", miles);
System.out.printf("%.2f hours to fly%n",   hoursFly);
System.out.printf("%.4f hours to drive%n", hoursDrive);
Output 
10.30 miles would take:
0.02 hours to fly
0.1717 hours to drive

Integer Formatting

A width number right-aligns integers in a fixed-width field, useful for aligning columns.

Right-aligning integers
final long KM_EARTH_TO_SUN   = 149598000L;
final long KM_SATURN_TO_SUN  = 1433449370L;

System.out.printf("Earth  is %12d km from the sun.%n", KM_EARTH_TO_SUN);
System.out.printf("Saturn is %12d km from the sun.%n", KM_SATURN_TO_SUN);
Output 
Earth  is    149598000 km from the sun.
Saturn is  1433449370 km from the sun.

String Formatting and Tables

A negative width (%-Ns) left-aligns a string. This is ideal for building aligned tables.

Formatted table
System.out.printf("%-10s | %-12s%n", "Dog age", "Human age");
System.out.printf("%-10s | %12s%n", "2 months", "14 months");
System.out.printf("%-10s | %12s%n", "6 months", "5 years");
System.out.printf("%-10s | %12s%n", "1 year",   "15 years");
Output 
Dog age    | Human age
2 months   |    14 months
6 months   |      5 years
1 year     |     15 years

Flushing Output

Output is buffered for performance — the system waits to collect characters before sending them. System.out.flush() forces the buffer to send immediately. In practice, println() and printf() with %n trigger automatic flushing on most Java implementations.
