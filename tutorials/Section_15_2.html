What Is printf()?

Sometimes println() isn't enough. Imagine you want to print a table where all the numbers line up in neat columns, or you want to show a price as exactly $3.50 instead of $3.5. That's what printf() is for — it lets you control exactly how your output looks.

System.out.printf() works by giving it a format string. The format string is like a template with blank spots (called format specifiers) that start with %. Java fills in each blank with the values you provide after the format string.

Basic syntax
System.out.printf("%-10s %5d %8.2f%n", "Item", 3, 19.99);
//                  ^       ^    ^  ^
//              left-pad  int  float  newline
Output 
Item            3    19.99

Format Specifier Cheat Sheet

Each specifier tells Java what type of value goes in that spot:

  • %d — a whole number (int or long)
  • %f — a decimal number (double or float)
  • %s — a String (any text)
  • %c — a single character
  • %b — a boolean (true or false)
  • %e — a number in scientific notation (like 1.23e+02)
  • %n — a newline (like hitting Enter) — use this instead of \n inside printf

Controlling Decimal Places

Put a dot and a number before f to set how many decimal places to show. For example, %.2f means "show 2 decimal places." You can also add a width before the dot — for example, %8.2f means "use 8 total characters, with 2 decimal places."

Decimal number examples
double miles = 10.3;
double hoursFly   = miles / 500.0;
double hoursDrive = miles / 60.0;

System.out.printf("%.2f miles would take:%n", miles);
System.out.printf("%.2f hours to fly%n",   hoursFly);
System.out.printf("%.4f hours to drive%n", hoursDrive);
Output 
10.30 miles would take:
0.02 hours to fly
0.1717 hours to drive

Aligning Numbers in Columns

Put a number before d to reserve that many characters for the number. Java right-aligns the number in that space, so all your numbers line up on the right edge — perfect for tables.

Right-aligning integers
final long KM_EARTH_TO_SUN   = 149598000L;
final long KM_SATURN_TO_SUN  = 1433449370L;

System.out.printf("Earth  is %12d km from the sun.%n", KM_EARTH_TO_SUN);
System.out.printf("Saturn is %12d km from the sun.%n", KM_SATURN_TO_SUN);
Output 
Earth  is    149598000 km from the sun.
Saturn is  1433449370 km from the sun.

Left-Aligning Strings in a Table

A minus sign before the width number (%-10s) left-aligns the text instead. This is how you build a nicely aligned table where text lines up on the left and numbers on the right.

Formatted table
System.out.printf("%-10s | %-12s%n", "Dog age", "Human age");
System.out.printf("%-10s | %12s%n", "2 months", "14 months");
System.out.printf("%-10s | %12s%n", "6 months", "5 years");
System.out.printf("%-10s | %12s%n", "1 year",   "15 years");
Output 
Dog age    | Human age
2 months   |    14 months
6 months   |      5 years
1 year     |     15 years

One Small Note About Flushing

Java sometimes saves up output before sending it to the screen (for speed). If your output doesn't appear right away, you can call System.out.flush() to force it out immediately. In practice, println() and printf() with %n usually handle this for you automatically.
